# The Complete Developer

This repository contains the code and documentation result of studying Martin Krause's book “The Complete Developer”.

### Chapter 1

Explains the basic Node.js enviroment, it's module ecosystem, and set up a facto standard server for full-stack development with Express.js

### Chapter 2

Covered how to use JS modules to create packages, import/export code, different ways to declared variables (and consts), arrow functions, asynchornous code, and other JS features native features.
